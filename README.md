# Parameterized N-Bit ALU in Verilog HDL

## Project Overview

This project implements a **Parameterized N-Bit Arithmetic Logic Unit (ALU)** using **Verilog HDL** and structural design techniques. The ALU supports multiple arithmetic, logical, and bit manipulation operations and generates processor-style status flags.

The design is fully parameterized, allowing the bit width to be changed easily by modifying a single parameter value. The ALU was developed and verified using **Vivado Design Suite**.

---

## Features

### Parameterized Architecture

* Configurable bit width using Verilog parameters.
* Default implementation uses an 8-bit datapath.
* Easily scalable to 16-bit, 32-bit, or higher widths.

### Structural Design

* Custom Full Adder implementation using gate-level modeling.
* Ripple Carry Adder constructed using Full Adder modules.
* Modular and reusable design hierarchy.

### Status Flags

The ALU generates the following processor-style flags:

| Flag     | Description                                                                 |
| -------- | --------------------------------------------------------------------------- |
| Carry    | Indicates carry-out during addition and borrow condition during subtraction |
| Overflow | Detects signed arithmetic overflow                                          |
| Zero     | Set when the result equals zero                                             |
| Sign     | Indicates the sign of the result (MSB)                                      |
| Parity   | Indicates even parity of the output result                                  |

## Supported Operations

| Select (S) | Operation      | Description                     |
| ---------- | -------------- | ------------------------------- |
| 000        | NAND           | Bitwise NAND operation          |
| 001        | XOR            | Bitwise Exclusive OR            |
| 010        | ADD            | Addition (A + B)                |
| 011        | INVERTER       | Bitwise NOT of A                |
| 100        | LEFT SHIFT     | Logical left shift by one bit   |
| 101        | RIGHT SHIFT    | Logical right shift by one bit  |
| 110        | 2'S COMPLEMENT | Generates two's complement of A |
| 111        | SUBTRACT       | Subtraction (A − B)             |

## Design Architecture

The ALU consists of the following modules:

### 1. Full Adder

A gate-level Full Adder designed using:

* XOR Gates
* AND Gates
* OR Gates

Outputs:

* Sum
* Carry Out

### 2. Ripple Carry Adder

Constructed using multiple Full Adder instances through a generate loop.

Features:

* Parameterized width
* Modular design
* Used for both addition and subtraction operations

### 3. Multiplexer

An 8-to-1 multiplexer selects the desired ALU operation based on the select input `S`.

### 4. Flag Generation Unit

Generates:

* Carry Flag
* Overflow Flag
* Zero Flag
* Sign Flag
* Parity Flag
